# Execute in develop

  1. Clone the repository
  2. Execute
  ```
  yanr install
  ```
  3. Have Nest Cli installed
  ```
  npm i -g @nestjs/cli
  ```
  4. Start the database
  ```
  docker-compose up -d
  ```

  ## Used Stack
    * MongoDB
    * NestJs
